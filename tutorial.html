Equações quadráticas

e sua resolução

Neste artigo, você aprenderá a resolver equações de segundo grau completas e incompletas, dos coeficientes ao conjunto solução.

Modelo de uma equação quadrática

Uma equação quadrática sempre terá o seguinte formato:

\(ax^2+bx+cx+c=0\)

Coeficientes

Antes de resolver uma equação quadrática é necessário encontrar seus coeficientes, pois a fórmula de resolução das equações de segundo grau é baseada nos coeficientes:

• Os coeficientes são \(a\), \(b\) e \(c\);

• O \(a\) sempre estará acompanhando o \(x^2\);

• O \(b\) sempre estará acompanhando o \(x\);

• O \(c\) sempre estará desacompanhado;

• Se um coeficiente não aparecer, o valor dele será 0.

Encontrar o \(Δ\)

O primeiro passo para resolver uma equação de segundo garu é encontrar o valor do \(Δ\). Para isto, utiliza-se a seguinte fórmula:

\(Δ=b^2-4ac\)

Observação

Caso o \(Δ\) seja negativo, a equação não tem resolução; caso seja igual a 0, a equação tem apenas um resposta; caso seja positivo, a equação tem duas soluções.

Encontrar o valor de \(x\)

Para encontrar o valor de \(x\), utiliza-se a seguinte fórmula:

\(x={-b\pm\sqrt{Δ}\over2a}\)

Finalização

Deste modo, resolve-se uma equação quadrática.
